## Complex Hotel Enterprise Network Architecture and Design Project

The Complex Hotel Enterprise Network Architecture and Design Project presents a comprehensive, scalable, and secure network infrastructure tailored for a modern hotel environment. The design supports multiple operational departments including IT, Management, Reception, Security, Restaurant, HR, Finance, Service Staff, Guest Rooms, and Conference Rooms, ensuring efficient communication, high availability, and seamless service delivery across the organization.

The network architecture is built to meet key business requirements such as continuous service availability, high-quality guest experience, operational efficiency, secure connectivity, reliable internal communication, and scalability for future expansion. To achieve redundancy and minimize downtime, the design incorporates dual ISPs with automatic failover mechanisms. Redundant core and access layer devices further enhance reliability and fault tolerance.

Network segmentation is implemented using VLANs to logically separate departmental traffic (VLAN 10–100), including a dedicated guest network and isolated CCTV network for enhanced security. A Next-Generation Firewall enforces strict access control policies, intrusion prevention, and traffic monitoring to safeguard sensitive data and ensure secure internet access. Inter-VLAN routing is handled by Layer 3 switches to maintain high performance and efficient internal communication.

The wireless infrastructure includes in-room access points, outdoor coverage for areas such as pools and gardens, and a centralized wireless controller to enable seamless roaming without connection drops. Power over Ethernet (PoE) switches support access points, IP phones, and CCTV cameras, simplifying deployment and cable management.

Core hardware components include routers, Layer 3 switches, access layer switches, a DHCP server for dynamic IP management, a wireless controller, Next-Generation Firewall, IP phones, printers, CCTV cameras, and both Fast and Gigabit Ethernet cabling for optimal performance. VoIP services are deployed across all departments and guest rooms to ensure reliable communication.

Overall, this project delivers a robust, high-performance enterprise network solution designed to support modern hotel technologies while maintaining security, scalability, and exceptional user experience.
